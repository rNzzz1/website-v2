# website-v2
# Портфолио Овсепа Узуняна

Многостраничное портфолио студента-разработчика с современным дизайном.

## Структура проекта

- `index.html` - Главная страница
- `pages/projects.html` - Страница проектов
- `pages/diary.html` - Учебный дневник
- `pages/contacts.html` - Контакты
- `styles/modern-styles.css` - Стили
- `scripts/main.js` - JavaScript
- `images/` - Изображения проектов
- `assets/` - Дополнительные файлы

## Особенности

- Современный адаптивный дизайн
- Анимации и плавные переходы
- Фильтрация проектов
- Интерактивный дневник прогресса
- Валидация форм
- Модальные окна

## Технологии

- HTML5
- CSS3 (Grid, Flexbox, Custom Properties)
- JavaScript (ES6+)
- Адаптивный дизайн
